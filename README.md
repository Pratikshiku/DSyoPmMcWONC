### 三、功能描述
系统分为（身份）：普通用户（乡村志愿者）、老人、医疗人员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 文娱活动
5. 生活照料
6. 土地
7. 个人中心
8. 生活照料订单
9. 土地承包



**老人功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 文娱活动管理
6. 活动报名管理
7. 医疗保健管理
8. 健康档案管理
9. 生活照料管理
10. 生活照料订单管理
11. 土地管理
12. 土地承包管理


**医疗人员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 文娱活动管理
6. 活动报名管理
7. 医疗保健管理
8. 健康档案管理
9. 生活照料管理
10. 生活照料订单管理
11. 土地管理
12. 土地承包管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 老人管理
5. 医疗人员管理
6. 乡村志愿者管理
7. 文娱活动管理
8. 活动报名管理
9. 医疗保健管理
10. 健康档案管理
11. 生活照料管理
12. 生活照料订单管理
13. 土地管理
14. 土地承包管理
15. 基础数据管理（医疗保健类型管理、文娱活动类型管理、身体情况管理、生活照料类型管理、土地类型管理）
16. 轮播图信息
